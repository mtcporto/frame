Credits

Frame is a curatorial project. Every film belongs to its creators. Every tool used to build this platform is open source. Here is a full accounting of both.

Film rights & sources

All films displayed on Frame are the intellectual property of their respective creators, studios, and rights holders. Frame does not host video files — all playback is done via embedded YouTube and Vimeo players directly from the originating channels.

Platform

Frame is built with plain HTML, CSS, and JavaScript — no frameworks, no bundlers, no dependencies beyond Plyr. The catalog is a hand-authored JSON file. Structured data follows Schema.org VideoObject, BreadcrumbList, and ItemList vocabularies.

The source code for this project is not currently public, but is intended for eventual open release.

Rights notice: If you are a filmmaker or rights holder whose work appears here and you would like it removed, or if you would like to update attribution, please contact us via the Vercel project and we will act promptly.
